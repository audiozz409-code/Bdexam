How SSC GPA is Calculated in Bangladesh

The Secondary School Certificate (SSC) examination is conducted by the Bangladesh Education Board. GPA is calculated on a 5.00 scale using the grading system implemented by the Ministry of Education.

  • A+ (Excellent): 80–100 marks → Grade Point 5.00
  • A (Very Good): 70–79 marks → Grade Point 4.00
  • A- (Good): 60–69 marks → Grade Point 3.50
  • B (Satisfactory): 50–59 marks → Grade Point 3.00
  • C (Acceptable): 40–49 marks → Grade Point 2.00
  • D (Pass): 33–39 marks → Grade Point 1.00
  • F (Fail): Below 33 marks → Grade Point 0.00
  • Optional 4th subject: Only included if it improves your GPA

University CGPA Scale in Bangladesh

Most public and private universities in Bangladesh use a 4.00 CGPA scale. Dhaka University, BUET, and other top institutions follow this system.

  • A+ / A = 4.00 (80–100%)
  • A- = 3.70 (75–79%)
  • B+ = 3.30 (70–74%)
  • B = 3.00 (65–69%)
  • B- = 2.70 (60–64%)
  • C+ = 2.30 (55–59%)
  • C = 2.00 (50–54%)
  • D = 1.00 (45–49%)
  • CGPA 3.50+ is typically required for government job merit consideration
